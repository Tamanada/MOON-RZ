# 🌙 MOON-RZ Academy

**La plus grande académie Web3 gamifiée — Learn · Practice · Earn.**

Une application éducative crypto qui transforme un débutant complet en utilisateur autonome du Web3,
à travers 10 académies, un token virtuel **MOON-RZ**, un système RPG et un simulateur de trading en temps réel.

## ✨ Fonctionnalités (v0.3)

- **10 académies complètes** — 60 modules de cours + 10 examens (Blockchain, Sécurité, DeFi, NFT,
  Web3 Entrepreneur, Trading, Advanced Trading, AI & Crypto, Tokenization, MOON-RZ Mastery)
- **Système RPG** — XP, 100 niveaux, 11 rangs (Crypto Explorer → MOON-RZ Legend), animations de level-up
- **Token MOON-RZ** (virtuel) — gagné en apprenant, badges NFT débloqués aux examens
- **Simulateur de trading** — 10 000 $ virtuels, prix crypto en **direct** (Binance), achat/vente spot, P&L temps réel
- **Missions quotidiennes** + streak

## 🚀 Lancer en local

C'est une app statique, sans build :

```bash
# avec Python
python -m http.server 5599
# puis ouvre http://localhost:5599
```

Ou ouvre simplement `index.html` dans le navigateur.

## 🛠️ Stack

HTML / CSS / JavaScript (un seul fichier `index.html`, zéro dépendance).
Persistance via `localStorage`. Migration prévue vers Vite + React + Supabase pour les comptes cloud.
